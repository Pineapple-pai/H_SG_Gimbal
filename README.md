# 🤖 机器人操作手册 (Operation Manual)

本文档详细说明了云台与底盘的控制模式、遥控器挡位映射以及键鼠操作逻辑。

## 1. 遥控器挡位说明 (Remote Controller)

控制逻辑基于 **DR16 接收机**。通过左右拨杆的组合切换不同的运行模式。

| 左拨杆 (Left SW) | 右拨杆 (Right SW) | 模式 (Mode)           | 说明 (Description)                                                    |
| :--------------- | :---------------- | :-------------------- | :-------------------------------------------------------------------- |
| **DOWN (下)**    | **DOWN (下)**     | **STOP (停止)**       | **整机失能**。云台无力，发射机构关闭，底盘锁定。 (断连保护默认此状态) |
| **MIDDLE (中)**  | **MIDDLE (中)**   | **KEYBOARD (键鼠)**   | **启用键盘鼠标控制**。底盘跟随云台，操作权移交至PC端。                |
| **UP (上)**      | **MIDDLE (中)**   | **VISION (视觉)**     | 视觉接管云台控制（自动瞄准），底盘保持运动能力。                      |
| **UP (上)**      | **UP (上)**       | **LAUNCH (发射)**     | **全自动开火**。视觉接管云台，摩擦轮开启，拨盘自动连发。              |
| **DOWN (下)**    | **MIDDLE (中)**   | **ROTATING (小陀螺)** | **底盘小陀螺模式**。底盘原地旋转，云台独立定向。                      |
| **MIDDLE (中)**  | **UP (上)**       | **FOLLOW (跟随)**     | **底盘跟随模式**。底盘正方向始终跟随云台朝向。                        |

> **注意：**
>
> - **视觉模式触发条件：** 除了拨杆组合外，**鼠标右键**按下也会强制激活视觉模式。
> - **视觉开火 (Vision Fire)：** 在视觉模式下，当视觉识别到目标（`vision_flag == true`）且鼠标双键（或左键）按下时，触发自动射击。

## 2. 键鼠操作指南 (Keyboard & Mouse)

只有当遥控器处于 **键鼠模式 (左中+右中)** 时，以下按键有效。

### 🕹️ 基础移动 (Movement)

| 按键      | 功能            | 逻辑实现               |
| :-------- | :-------------- | :--------------------- |
| **W**     | 前进 (Forward)  | 底盘 Y 轴速度 +        |
| **S**     | 后退 (Backward) | 底盘 Y 轴速度 -        |
| **A**     | 左移 (Left)     | 底盘 X 轴速度 +        |
| **D**     | 右移 (Right)    | 底盘 X 轴速度 -        |
| **Shift** | 加速 (Boost)    | 按住时触发底盘高速模式 |

### 🛠️ 战术技能 (Tactical Skills)

| 按键     | 功能                       | 说明                                                                                                               |
| :------- | :------------------------- | :----------------------------------------------------------------------------------------------------------------- |
| **Q**    | 开启小陀螺                 | 设定旋转速度为 220                                                                                                 |
| **E**    | 关闭小陀螺                 | 设定旋转速度为 0                                                                                                   |
| **X**    | **一键掉头 (Turn Around)** | 云台快速旋转180度（`gimbal.setTrueAround`）                                                                        |
| **C**    | 视觉/飞行模式切换          | **点击：** 切换视觉识别模式 (装甲板/小能量/大能量 循环切换)<br>**长按/生效：** 触发 `setFly(100)` (飞坡/飞行模式?) |
| **Ctrl** | 刷新/UI重置                | 发送 `UI_F5` 信号，同时重置飞行模式 (`setFly(0)`)                                                                  |

### 🔥 武器控制 (Weapon System)

| 按键            | 功能                | 说明                                         |
| :-------------- | :------------------ | :------------------------------------------- |
| **V**           | **开启发射机构**    | 摩擦轮启动，准备射击 (Enter `AUTO` State)    |
| **B**           | **关闭发射机构**    | 摩擦轮停止，禁止射击 (Enter `DISABLE` State) |
| **F**           | 增加功率上限        | 底盘功率限制 +10W                            |
| **G**           | 减少功率上限        | 底盘功率限制 -10W                            |
| **Mouse X**     | 云台 Yaw 旋转       | 鼠标左右移动控制云台左右转                   |
| **Mouse Y**     | 云台 Pitch 旋转     | 鼠标上下移动控制云台俯仰                     |
| **Mouse Left**  | **射击 (Fire)**     | 在开启状态（V键后）触发拨盘射击              |
| **Mouse Right** | **自瞄 (Auto-Aim)** | 按住激活视觉辅助锁定目标                     |

## 3. 状态机逻辑概览

### 🚀 发射机构 (Shoot FSM)

- **DISABLE**: 按下 `B` 键或遥控器处于 `STOP` 模式。
- **ONLY (单发)**: 视觉模式下，鼠标左右键同时按下（视觉辅助射击）。
- **AUTO (连发)**:
  1.  按下 `V` 键后，且非视觉射击状态。
  2.  遥控器处于 `LAUNCH` 模式 (右拨杆上)。
  3.  此时射频受热量控制算法 (`Heat_Control`) 限制。

### 🔭 云台 (Gimbal FSM)

- **DISABLE**: 遥控器 `STOP` 模式。
- **VISION**: 遥控器处于视觉挡位 OR 检测到鼠标右键，且视觉数据有效。
- **KEYBOARD**: 遥控器处于键鼠挡位（且未触发视觉）。
- **NORMAL**: 其他普通遥控模式。
