# 操作手键位表

> 仅保留键鼠模式，基于当前仓库代码整理。
> 主要参考：
> - `Task/RemoteSwitch/RemoteSwitchTask.cpp`
> - `APP/Mod/RemoteControl/DR16Controller.hpp`

## 1. 键鼠模式

进入条件：

- 左拨杆中位
- 右拨杆中位

### 1.1 云台/视觉

| 输入 | 功能 |
| --- | --- |
| 鼠标移动 X | 云台 Yaw 控制 |
| 鼠标移动 Y | 云台 Pitch 控制 |
| 鼠标右键 | 进入视觉模式 |
| 鼠标左键 | 开火 |
| 鼠标右键 + 左键 | 视觉开火模式 |

### 1.2 底盘移动

| 输入 | 功能 |
| --- | --- |
| `W` | 前进 |
| `S` | 后退 |
| `A` | 左移 |
| `D` | 右移 |

### 1.3 底盘/系统功能

| 输入 | 功能 |
| --- | --- |
| `C` 单击 | 切换底盘跟随开关 |
| `X` 长按 | 小陀螺 |
| `Shift` 长按 | 发送 `Shift` 状态给底盘 |
| `V` 松开沿 | 功率 +10 |
| `B` 松开沿 | 功率 -10 |
| `Ctrl` 单击 | 刷新 UI / `UI_F5` |

### 1.4 发射机构

| 输入 | 功能 |
| --- | --- |
| `R` 单击 | 开启摩擦轮 |
| `G` 单击 | 关闭摩擦轮 |
| 鼠标左键 | 手动开火 |
| 鼠标右键 + 左键 | 视觉开火模式 |

## 2. 备注

- 当前代码中，键鼠模式下发射机构的开关由 `R/G` 控制。
- 当前代码中，视觉模式和视觉开火模式是两回事：
  - 视觉模式：云台使用视觉目标
  - 视觉开火模式：发射机构允许视觉触发开火
- 如果后续你改了 `RemoteSwitchTask.cpp` 或 `DR16Controller.hpp`，这份表也需要同步更新。
