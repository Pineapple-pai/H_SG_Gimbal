# 步兵云台与发射机构排障清单

## 联调顺序

1. 先查供电、电机使能、急停和保护状态
2. 再查 CAN/UART 是否在线
3. 再查 IMU 数据是否连续、频率是否正常
4. 再查电机方向、反馈方向、限位方向
5. 最后再做闭环调参

## 云台抖动

现象：

- 静止小幅抖动
- 大角度回中时振荡
- 开视觉后抖动明显

优先检查：

- IMU 噪声和滤波
- 角度环、速度环频率
- 电机方向和反馈方向
- 视觉给定是否跳变
- 微分或阻尼是否过弱

## 云台反向

优先检查：

- 电机正负方向定义
- IMU pitch/yaw 方向定义
- 目标角和反馈角是否同号
- 减速比换算后符号是否被翻转

## 摩擦轮转速异常

优先检查：

- 左右轮目标值是否一致
- PID 参数是否差异过大
- 限流或功率限制是否介入
- 启动时是否需要转速斜坡

## 拨弹卡弹

优先检查：

- 单发目标角度是否正确
- 卡弹判据阈值是否过敏或过钝
- 反卡弹是否只反转不复位
- 摩擦轮未起转时是否错误拨弹

## IMU 异常

优先检查：

- 串口接收频率
- 数据包完整性
- 断联计时
- yaw 连续角处理
- 零漂和安装方向
