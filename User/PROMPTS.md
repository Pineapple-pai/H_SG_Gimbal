# RoboMaster 电控提问模板

这份文件用于快速复用提问模板。
默认适用于当前步兵电控工程和 `robomaster-electrical-control` skill。

默认硬件背景：

- 主控：STM32F407
- 云台电机：yaw DN4310，减速比 1:10
- 云台电机：pitch DN4310，减速比 1:10
- 摩擦轮电机：M3508
- 拨弹电机：M2006，减速比 1:36
- IMU：HI12H3，250 Hz

## 通用模板

```text
帮我处理 [问题/目标]。

按 robomaster-electrical-control skill 流程来：
1. 先说明你的假设与约束
2. 做问题定位
3. 列出拟修改文件
4. 给出拟修改代码或 diff
5. 等我确认后再实际修改

要求：
- 不要直接改代码
- 新增注释用中文
- 注意不要把原中文注释改乱码
```

## 云台问题模板

```text
帮我分析云台 [yaw/pitch] 的 [抖动/超调/反向/跟随慢]。

按 robomaster-electrical-control skill 流程来：
1. 先基于当前工程做问题定位
2. 重点检查方向、减速比、IMU 250Hz 使用、控制周期
3. 如果需要改代码，先给我拟修改代码
4. 等我确认后再改
```

## 自瞄与视觉联调模板

```text
帮我分析自瞄接入后云台 [抖动/延迟大/丢目标后乱转] 的问题。

按 robomaster-electrical-control skill 流程来：
1. 先区分是视觉链路问题还是底层控制问题
2. 说明你当前的假设
3. 列出可能涉及的文件
4. 给出拟修改方案和代码
5. 等我确认后再改
```

## 摩擦轮问题模板

```text
帮我优化摩擦轮的 [启动慢/转速波动/左右轮不一致]。

按 robomaster-electrical-control skill 流程来：
1. 先分析控制链路和可能原因
2. 给出调参建议
3. 如果需要改代码，先列文件和拟修改代码
4. 等我确认后再改
```

## 拨弹与卡弹问题模板

```text
帮我检查拨弹机构的 [单发不准/连发异常/卡弹]。

按 robomaster-electrical-control skill 流程来：
1. 先检查减速比换算、单发角度、状态机逻辑
2. 再给出问题定位
3. 如需修改，先给我看拟修改代码
4. 我确认后你再实际修改
```

## 纯调参模板

```text
根据当前工程，给我一个 [yaw/pitch/摩擦轮/拨弹] 的调参顺序。

要求：
- 不改文件
- 结合当前代码结构回答
- 按“先查什么，再调什么，观察什么现象，怎么判断通过”来写
```

## 代码审查模板

```text
帮我审一下这个模块：[模块名]。

要求：
- 重点找 bug、风险、行为回归
- 重点看步兵云台、摩擦轮、拨弹机构相关逻辑
- 不要直接改代码
- 如果建议修改，先给出拟修改代码，等我确认
```
