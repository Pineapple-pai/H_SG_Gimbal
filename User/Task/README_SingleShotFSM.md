# 单发有限状态机 (Single Shot FSM) 设计与使用说明

本文档详细说明了在发射机构任务 (`ShootTask`) 中新增的单发控制逻辑，包括单击检测与停火保护机制。

## 1. 核心功能

该模块主要实现了以下功能：

- **精确单发控制**：每次扣动扳机（鼠标左键或视觉触发），拨盘严格转动一颗子弹的角度（40度）。
- **输入防抖与长按分离**：通过独立的状态机区分“短按单击”和“长按”，防止误触发。
- **智能停火保护**：在单发动作完成后，通过电流检测和超时机制实现了“自动刹车”，防止拨盘过转或电机过载。

## 2. 状态机架构

系统引入了两个新的有限状态机 (FSM) 配合原有的发射状态机工作：

### 2.1 单击有限状态机 (`Class_ClickFSM`)

负责处理输入信号（鼠标/遥控器/视觉），判定是否为有效单击。

- **状态列表**:
  - `CLICK_DISABLE`: 失能/初始状态。
  - `PRESS_DOWN`: 按键按下状态。
  - `CLICK`: **有效单击**（按键按下后在 200ms 内松开）。
  - `LONG_PRESS`: 长按状态（按键按下超过 200ms），不触发单击。
  - `RELEASE`: 按键松开状态（等待下一次按下）。
- **逻辑流程**:
  1.  检测到按键按下 -> 进入 `PRESS_DOWN`。
  2.  如果松开按键：
      - 按住时间 < 200ms -> 判定为 `CLICK` (触发开火)。
      - 按住时间 > 200ms -> 判定为 `RELEASE` (忽略操作)。
  3.  如果一直按住 -> 超过 200ms 切换至 `LONG_PRESS`。

### 2.2 停火有限状态机 (`Class_StopFireFSM`)

负责在单发指令发出后，监控电机状态并执行停火逻辑。

- **状态列表**:
  - `STOP_FIRE_DISABLE`: 失能/空闲状态。
  - `STOP_FIRE_ACTIVE`: **激活状态**（正在执行单发动作）。
  - `STOP_FIRE_PROCESSING`: **处理状态**（触发刹车）。
- **逻辑流程**:
  1.  收到开火指令 -> 进入 `STOP_FIRE_ACTIVE`。
  2.  监控拨盘电机电流与时间。
  3.  **触发条件**: 电流 > 8000 (阻力变大，子弹上膛或卡滞) **或** 时间 > 1秒 (超时保护)。
  4.  满足条件 -> 进入 `STOP_FIRE_PROCESSING`，锁定当前角度（刹车）。
  5.  完成刹车 -> 回到 `STOP_FIRE_DISABLE`。

## 3. 整体控制逻辑 (`Class_ShootFSM`)

在 `Booster_Status::ONLY` (单发模式) 下的控制流：

1.  **输入采集**: 获取鼠标左键状态和视觉开火信号 (`fire_flag`)。
2.  **输入处理**: 将信号送入 `ClickFSM` 更新状态。
3.  **开火触发**:
    - 如果 `ClickFSM` 变为 `CLICK` 状态 **且** 热量限制允许 (`HeatLimit`)：
    - 拨盘目标角度减少 40度 (`Dail_target_pos -= 40.0f`)。
    - 激活 `StopFireFSM`。
4.  **过程监控**:
    - `StopFireFSM` 持续监控电机。
    - 如果 `StopFireFSM` 进入 `PROCESSING` 状态 -> 强制将目标角度设为当前角度 (`Dail_target_pos = current_angle`)，实现急停。

## 4. 参数配置

在 `ShootTask.hpp` 和 `ShootTask.cpp` 中可调整的参数：

- `click_time_threshold`: 单击判定时间阈值 (默认 200ms)。
- `stop_torque_threshold`: 停火电流/力矩阈值 (默认 8000)。
- `stop_time_threshold`: 单发最大动作时间 (默认 1秒)。
- 拨盘角度步长: 默认 40度 (基于 9槽拨盘: 360/9)。

## 5. 验证方法

1.  **单击测试**: 快速点击并松开鼠标左键，观察拨盘是否准确转动约40度并停止。
2.  **长按测试**: 按住鼠标左键不动，拨盘应不动作（或仅动作一次后停止，取决于初始状态），松开后不应再次动作。
3.  **停火/堵转测试**:
    - 正常发射时，观察电流波形，确认发射后电流迅速归零。
    - 人为制造阻力（小心操作），观察是否在电流升高后迅速触发刹车逻辑。
