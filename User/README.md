# 🤖 机器人操作手册 (Operation Manual)

**当你点开这份手册的时候，你已经成为舵轮的主人了** **_<span style="color: red">!!</span>请谨慎驾驶<span style="color: red">!!</span>_**
本文档详细说明了云台与底盘的控制模式、遥控器挡位映射以及键鼠操作逻辑。

## 1. 遥控器挡位说明 (Remote Controller)

控制逻辑基于 **DR16 接收机**。通过左右拨杆的组合切换不同的运行模式。

| 左拨杆 (Left SW) | 右拨杆 (Right SW) | 模式 (Mode)            | 说明 (Description)                                                               |
| :--------------- | :---------------- | :--------------------- | :------------------------------------------------------------------------------- |
| **DOWN (下)**    | **DOWN (下)**     | **STOP (停止)**        | **整机失能**。云台无力，发射机构关闭，底盘锁定。 (断连保护默认此状态)            |
| **MIDDLE (中)**  | **DOWN (下)**     | **FOLLOW (跟随)**      | **底盘跟随模式**。底盘正方向始终跟随云台朝向。                                   |
| **DOWN (下)**    | **MIDDLE (中)**   | **ROTATING (小陀螺)**  | **底盘小陀螺模式**。底盘原地旋转，云台可独立定向/视觉瞄准。                      |
| **MIDDLE (中)**  | **MIDDLE (中)**   | **KEYBOARD (键鼠)**    | **启用键盘鼠标控制**。底盘跟随云台，操作权移交至PC端。                           |
| **DOWN (下)**    | **UP (上)**       | **LAUNCH (发射/视觉)** | **增强发射模式**。波轮控制手动打弹；**若收到视觉开火信号则自动全力(25Hz)开火**。 |
| **MIDDLE (中)**  | **UP (上)**       | **RUNE (符节/大能量)** | **符节模式**。完全交由视觉控制瞄准与射击。                                       |
| **UP (上)**      | **UP (上)**       | **ROTATING (小陀螺)**  | **备用小陀螺**。功能同左下右中。                                                 |

> **注意：**
>
> - **视觉模式触发条件：** 除了拨杆组合外，**鼠标右键**按下也会强制激活视觉模式。
> - **视觉开火 (Vision Fire)：** 在视觉模式下，当视觉识别到目标（`vision_flag == true`）且鼠标双键（或左键）按下时，触发自动射击。
>
> **⚠️ 左手拨轮 (SW) 说明：**
> 在 **LAUNCH (发射)** 或 **VISION (视觉)** 模式下：
>
> - **SW 上拨 (Down/负值)**：**单发模式** (Single Shot)，拨一次开一枪。
> - **SW 下拨 (Up/正值)**：**连发模式** (Auto Fire)，上拨角度控制射频 (Hz)。

## 2. 键鼠操作指南 (Keyboard & Mouse)

只有当遥控器处于 **键鼠模式 (左中+右中)** 时，以下按键有效。

### 🕹️ 基础移动 (Movement)

| 按键      | 功能            | 代码行为（当前）       |
| :-------- | :-------------- | :--------------------- |
| **W**     | 前进 (Forward)  | `LY = -1`              |
| **S**     | 后退 (Backward) | `LY = +1`              |
| **A**     | 左移 (Left)     | `LX = +1`              |
| **D**     | 右移 (Right)    | `LX = -1`              |
| **Shift** | 加速 (Boost)    | 按住有效，`Shift = 1`  |

### 🛠️ 战术技能 (Tactical Skills)

| 按键     | 功能                       | 说明                                                                                                   |
| :------- | :------------------------- | :----------------------------------------------------------------------------------------------------- |
| **X (长按)** | 小陀螺                   | 长按生效，按住且超过长按阈值后发送 `Rotating_vel = 220`，松开后发送 `Rotating_vel = 0`                |
| **C**    | **一键掉头 (Turn Around)** | 点击触发 `gimbal.setTrueAround()`，云台约 500ms 内完成快速转向                                          |
| **Ctrl** | 刷新/UI重置                | 点击触发 `UI_F5`，并执行 `setFly(0)`                                                                    |
| **V**    | 功率 +10                   | **松开瞬间触发**（下降沿）`setPower(+10)`                                                                |
| **B**    | 功率 -10                   | **松开瞬间触发**（下降沿）`setPower(-10)`                                                                |

### 🔫 攻击操作 (Combat)

| 按键             | 功能                       | 说明                                                                 |
| :--------------- | :------------------------- | :------------------------------------------------------------------- |
| **R**            | **开启发射 (Enable Fire)** | 点击后 `booster_enabled = true`                                      |
| **G**            | **关闭发射 (Disable)**     | 点击后 `booster_enabled = false`                                     |
| **鼠标左键**     | **射击 (Fire)**            | 发射使能后生效；`ONLY` 模式下短按单发，按住超过 1s 自动转 15Hz 连发 |
| **鼠标右键**     | 视觉瞄准触发               | 右键会触发 `isVisionMode()`；有目标时云台进入视觉状态                |
| **右键 + 左键**  | 视觉开火触发               | 触发 `isVisionFireMode()`，进入视觉开火逻辑                          |

### 📌 备注

- 云台在 `KEYBOARD` 状态下当前读取的是 `remoteRight` 通道（右摇杆），鼠标速度输入代码已注释。
- 上述说明以当前代码为准（`Task/RemoteSwitch/RemoteSwitchTask.cpp`、`Task/ShootTask.cpp`、`APP/Mod/RemoteControl/DR16Controller.hpp`）。
